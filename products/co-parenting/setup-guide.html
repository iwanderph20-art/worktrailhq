👨‍👩‍👧 Co-Parenting Calendar
Shared Custody Calendar · Setup Guide

Thanks for your purchase! This is a single file (co-parenting.html) that runs in any browser — no install, no account, no subscription. Everything you enter is stored privately on your own device. Here's how to get set up in a few minutes.

1 · Set up your family

1
Open the app.

Double-click co-parenting.html — it opens in your browser.

2
Open Setup.

Tap ⚙ Setup (top-right). Name your calendar (e.g. "The Rivera Kids").

3
Name both parents & pick colors.

Enter a name for Parent A and Parent B (Mom / Dad / first names — whatever you use) and choose a color for each. Those colors run through the whole calendar so it's always clear whose day is whose.

4
Add your kids.

Tap ＋ Add a child for each child and give them a name. Tap the colored dot to change their color. Events can then be tagged to each child.

2 · Fill your custody schedule

In ⚙ Setup → Custody schedule, choose the rotation your family uses, set the start date, pick who starts, choose how far ahead to generate, and tap Fill schedule from this pattern. The calendar fills in automatically. You can always tap any single day to change it.

PatternHow it works
Week on / week offOne parent for a full week, then the other.
2-2-32 days A, 2 days B, 3 days A — then it flips. Weekends alternate every two weeks.
2-2-5-52 days each, then 5 days each. Predictable weekdays.
3-4-4-3A balanced near-50/50 split across a two-week cycle.
Every other weekendOne home base parent, the other has alternating weekends.
Tip. For the weekend patterns, set the start date to a Monday so the weekends land where you expect. Not sure which day starts a cycle? Fill it, glance at the calendar, and if it's a week off just re-run it with the other starting parent.

3 · Add events & handoffs

Tap any day, then ＋ Add event. Give it a name, pick a type (Handoff, Activity, School, Medical, Expense, Other), set a time (or mark it all-day), tag the child, choose who's responsible, and add a location and notes. Choose Expense to record a cost and how it splits. Everything shows up on the calendar and in the Agenda view.

Share a read-only view with the other parent

The other parent can see your calendar on their own phone or computer — without being able to change it. Open ⚙ Setup → Share a read-only view and tap Create a read-only link. Copy the link and send it (text, email, whatever). When they open it, they see your calendar, custody days, and events in a clean read-only view — the setup and editing controls simply aren't there.

How it works. The link carries a private, compressed snapshot of your calendar inside the web address itself — nothing is uploaded to any server, and there's no account for either of you. Because it's a snapshot, it shows the calendar as it was when you made the link: when your schedule changes, just create and send a fresh link. (Your uploaded photo isn't included, so the link stays short.)

For sharing to work, the file must be hosted (see "Use it on your phone" below) so the link opens on any device — a link made from a file opened directly by double-click only works on that same computer.

Keeping the other parent in the loop (new events)

Want new events to reach your co-parent automatically? Open ⚙ Setup → Notify the other parent and paste a free delivery URL. Each new event you save is then sent there (and you can hit Send test to confirm it arrives).

  • Email it — create a free endpoint at Web3Forms or Formspree and paste the URL. New events land in an inbox.
  • Log to a shared spreadsheet — use a Google Sheet + Apps Script Web App URL so both parents can see the running list.
  • Anywhere else — a Zapier or Make.com webhook can push events to a calendar, SMS, or another app.
Private by default. Nothing is uploaded unless you add a delivery URL. Without one, your schedule and events live only in this browser, on this device.

Use it on your phone

Host the file for free (drag it onto app.netlify.com/drop for an instant link), open the link on your phone, and choose Add to Home Screen for a one-tap, full-screen calendar. Add ?embed=1 to share a clean view without the setup controls, or ?theme=dark / ?theme=light to force a look.

Save & export

Everything saves automatically on your device. Use ⚙ Setup → Export events (CSV) to download a spreadsheet of every event (handy for records or a mediator), or Print for a paper copy of the month.

Please note. This is an organizing tool to help two households stay coordinated — it is not a legal custody agreement or legal advice. Follow your own parenting plan or court order.
